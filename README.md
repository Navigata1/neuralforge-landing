# NeuralForge.FM — Landing Page

Marketing and introduction site for [NeuralForge.FM](https://github.com/Navigata1/NeuralForge.FM),
the verification-gated AI workspace ("Living Workspace OS") where nothing executes on trust alone.

- Single static page (`index.html`), zero build step, zero dependencies.
- Product screenshots in `assets/` are compressed renders of the NeuralForge product reference library (screens 02, 04, 05, 06, 12, 26, 52).
- Deployed on Vercel as a static site.

The product itself lives in the main repo: https://github.com/Navigata1/NeuralForge.FM
